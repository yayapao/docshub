## 小程序配置

- [page.json](https://developers.weixin.qq.com/miniprogram/dev/reference/configuration/app.html) 页面配置文件，用于配置当前页面的窗口样式、页面标题等
- app.json 小程序全局配置文件，用于配置全局属性和页面路由
- project.config.json 项目配置文件，用于保存项目配置和开发者的个性化配置
- sitemap.json 配置小程序及其页面是否允许被微信索引，提升小程序在搜索引擎中的收录能力
