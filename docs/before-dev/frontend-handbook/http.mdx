## Headers

### Origin

`Request.Origin` 用来标识请求来源（Protocol, Hostname, Port(可选，默认为 80), null）。

- Null 表示请求来源为**隐私敏感**，或者一些特殊情况

添加 Origin 请求标头的情况：

- 跨域
- 同域下，除了 `GET` 和 `HEAD` 请求外，都会携带 Origin

因此，我们在服务端处理跨域时，需要设置 `Access-Control-Allow-Origin` 为 `Request.Origin`.
