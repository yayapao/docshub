## 时间转换

记录时间戳、time 与时间字符串之间的相互转换。

```go
// 时间戳转 time
time.Unix(1600000000, 0) // (秒, 纳秒)

// time 转时间戳
time.Now().Unix()

// 时间字符串转 time
time.Parse("2006-01-02 15:04:05", "2020-09-13 10:00:00") // (时间格式, 时间字符串)

// time 转时间字符串
time.Now().Format("2006-01-02 15:04:05")
```
