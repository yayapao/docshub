HTML5 作为一种 markup language，难点在于**如何在合适的场景使用对应的标签标记？** 因此，在学习过程中，我们首先需要丰富自己的“武器库”，同时能够了解各个“武器”的特性，学以致用。

我将 HTML5 看作一个综合体，浏览器、HTML、CSS 技术、JavaScript 都在其中扮演着重要的角色，其中 JavaScript 的使用范围和知识广度均达到一定程度，因此我将其单独归类。

在沉淀 HTML5 过程中，尤其是在学习浏览器相关模块时，需要和计算机网络的知识紧密结合，因为从某种意义上来说，浏览器是各种网络协议的代理。
