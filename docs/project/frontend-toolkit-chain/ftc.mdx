**FTC(Frontend Toolkit Chain)** 是我针对前端开发体验的探索，希望能够解决开发阶段的规范、标准化以及安全等诸多问题，最终输出高质量的构建产物。

## 如何理解工具链

工具链是一系列用于研发流程中的工具。

工具被用来解决在开发过程中让人感到“不爽”的点。不同工具的排列、组合形成“链”的概念。

最终的目的是抹平研发流程中的“摩擦点”，在输出可靠产物的前提下，提升开发效率。

## 如何在开发时介于质量管理

将**开发-发布**过程拆解出其关键的生命周期：

- 创建应用
  - 统一模版
  - 统一框架
- 搭建基础设施
  - 统一代码规范
  - 统一开发语言
- 开发
- 打包
- 发布
