## Installation

### Requirements

- Node.js 14.0.0+

Choose your favorite package manager:

```bash
# yarn(recomanded)
yarn global add joycra-scripts

# npm
npm i -g joycra-scripts

# pnpm
pnpm i -g joycra-scripts
```

## Usage

Once you have installed joylint in your project. You can exec `joycra-scripts --help` to get help informations.

**Create a JOY App**

```bash
joycra-scripts [your awesome app]
# or just do as it guides
joycra-scripts
```

**Check Environment values**

```bash
joycra-scripts --info
```

**Print joycra-scripts version**

```bash
joycra-scripts --version
```
