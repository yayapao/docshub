**FTE(Frontend Trustworthy Engineering)** 是我在 2020 年开始提出的一个构想，旨在通过一系列技术手段来确保 web 应用在浏览器运行时内的运行状态是安全、可靠、可感知的。

## 如何理解可信赖

可信赖是应用在运行时的一种状态描述。

## 如何选择指标

:::info
Metrics provide insight about how users are experiencing your application.
:::

指标的选择归根究底是对用户体验的量化。
