## How to self-hosted?

在进行本地开发前，我们需要搭建运行 Sentry 所需的服务和环境。

作为一个开源项目，Sentry 提供了官方的私有化部署方案，即[self-hosted](https://github.com/getsentry/self-hosted)，它是一个 composed 的容器镜像，如果你仅需要本地跑或者接入少量项目时，它是足够的。

但是，如果你希望拥有可扩展的能力并且能够支撑更多的项目和并发量，我更建议**对其服务进行拆分**，我会直接切入到该点对其进行介绍。
