## Requirements

- Node.js 8.0.0+

## Installation

Choose your favorite package manager:

```bash
# pnpm(recomanded)
pnpm add joylint -D

# npm
npm i joylint -D

# yarn
yarn add joylint -D
```

## Usage

Here's a quick demostration:

**.prettierrc.js**

```js
const joylintPrettier = require('joylint/dist/prettier')

module.exports = {
  ...joylintPrettier,
}
```

**.eslintrc.js**

:::note Supported
- eslint4React
- eslint4Vue2
- eslint4Vue3
:::

```js
// define customized options
const customizedOptions = {
  rules: {
    semi: ['error', 'always'],
  },
}

module.exports = {
  // choose what you want
  extends: [require.resolve('joylint/dist/eslint4React')],
  ...customizedOptions,
}
```

**tsconfig.js**

:::note Supported
- tsconfig4React
- tsconfig4Vue3
:::

```json
{
  "extends": "./node_modules/joylint/dist/reference/tsconfig4React",
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  },
  "include": ["src", "configs/**/*"]
}
```
