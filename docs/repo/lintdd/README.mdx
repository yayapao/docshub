## Requirements

- Node.js 8.0.0+

## Installation

Choose your favorite package manager:

```bash
# pnpm(recomanded)
pnpm add lintdd -D

# npm
npm i lintdd -D

# yarn
yarn add lintdd -D
```

## Usage

Here's a quick demostration:

**.prettierrc.js**

```js
const lintddPrettier = require('lintdd/dist/prettier')

module.exports = {
  ...lintddPrettier,
}
```

**.eslintrc.js**

```js
// define customized options
const customizedOptions = {
  rules: {
    semi: ['error', 'always'],
  },
}

module.exports = {
  // react + ts
  extends: [require.resolve('lintdd/dist/eslint4React')],
  ...customizedOptions,
}
```

**tsconfig.js**

```json
{
  "extends": "./node_modules/lintdd/dist/reference/tsconfig4React",
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  },
  "include": ["src", "configs/**/*"]
}
```
