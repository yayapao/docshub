## 使用自定义高亮组件

自定义高亮组件放置在 `src/components/Highlights` 内，在 `.mdx` 文件内通过相对路径进行引入

### Text

`HighlightWithText` 高亮普通文本

```tsx
import {
  HighlightWithText,
} from '../../../src/components/Highlights'

<HighlightWithText bgcolor="#f30" style={{ fontSize: '12px' }}>Some text content!</HighlightWithText>
```
